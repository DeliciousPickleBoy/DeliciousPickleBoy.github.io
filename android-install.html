Emulation

So first we need a stable, well maintained Android emulator that runs well on modern hardware. Thankfully, the Android devs have this sorted out with the apk emulator bundled inside Android Studio downloadable at developer.android.com/studio/.

Once you have this downloaded it is smart to check the file integrity with the provided SHA256 hash:
 Windows: Open Powershell and type the command "Get-FileHash "[path-to-file]" -Algorithm | Format-List" (without outer speech marks) and hit Enter. This should look something like this:

   PS C:\Users\Stulto> Get-FileHash "C:\Users\Stulto\Downloads\android-studio-2020.3.1.26-windows.exe" -Algorithm SHA256 | Format-List

   Algorithm : SHA256
   Hash      : D9181AE1668FC4A5F3A19AA5A2F9951F022BFF1359A70AA0F0E7987E248C740C
   Path      : C:\Users\Stulto\Downloads\android-studio-2020.3.1.26-windows.exe
 Linux, and MacOS: Open Terminal and type the command "shasum -a 256 "[path-to-file]"" (without outer speech marks) and hit Enter. This should look something like this:

   stulto@mac-n-cheese:~$ shasum -a 256 "/home/stulto/Downloads/android-studio-2020.3.1.26-mac.dmg"
   7da10ce0c3e998393045f6de3c37df46bd95e3bacb3f803d63fd85bc67148d6e  /home/stulto/Downloads/android-studio-2020.3.1.26-mac.dmg
Once you have the hash you can then check it against the one advertised on the download page to make sure the file has downloaded correctly. Remember, casing is not important. You can now start the installation keeping the default options. Once installation is compleate open Android Studio, when asked for permission to send telemetry select "Don't allow", and then accept any requested updates if you want. Continue through setup keeping default options.

To use the emulator, simply select More Actions (or the three stacked dots aka burger icon) > AVD Manager > Create Virtual Device... > Select the device you want to emulate and download the required image, once done, click Done > Finish > Install Haxm, you can also set the screen orientation, and disable the device frame if you so desire, once done, click Finish. Now drag and drop your APK to install it to the emulator.

To get your installed apps drag up from the bottom of the screen, and then hold and drag to place it on the home screen for quick access. To exit fullscreen mode simply press on the very top of the screen, you can then resize the windowed emulator to anything you want. This will also give you access to the emulator tool bar, which will allow you to change volume, take screenshots and more (if you want up-to-date shortcuts, just hover over the buttons (that'll show the tooltips ;) )).
